# 微信公众号

面向中文读者的 AI 科普与实践分享。

## 📱 公众号定位

降低 AI 使用门槛，提供实用的 AI 工具和方法，分享个人使用经验。

## 🎨 内容特点

### 与 GitHub Pages 文章的区别

| 特点 | GitHub Pages | 微信公众号 |
|------|-------------|-----------|
| 受众 | 开发者/技术人员 | 大众读者 |
| 深度 | 技术深度分析 | 科普+实践 |
| 长度 | 长文可拆分多期 | 控制在 2000-3000 字 |
| 语言 | 中英混杂 | 中文为主 |
| 排版 | 代码友好、公式完整 | 图文结合、适合手机阅读 |

### 内容适配原则

1. **科普化**：技术概念用通俗语言解释
2. **实用化**：每篇给出可直接操作的建议
3. **场景化**：用具体场景说明问题
4. **视觉化**：多用图表、截图辅助理解

## 📊 内容类型

- **热点解读**：最新 AI 新闻/工具/趋势解读
- **工具测评**：AI 工具使用体验和推荐
- **教程干货**：具体的 AI 使用方法
- **经验分享**：个人使用 AI 的心得
- **问答互动**：读者问题解答
